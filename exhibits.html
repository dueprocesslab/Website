Fast Add Template

When you add a new exhibit, copy one row and fill in these fields:

  • Exhibit # (Ex. 12, Ex. 35A, etc.)
  • Date (use the document date or the capture date)
  • Standardized File Name (keep consistent)
  • Related Issue (Issue # + short topic)
  • Agency Relevance (tags)
  • Public Summary (2–6 sentences, neutral, factual)
